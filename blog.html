1. What is DOM and what is the purpose of DOM?

Ans: DOM mean Document Object Model. It think every html element as object. DOM used to modify html document dynamically. We can manipulate javascript in html by DOM object. Every html tag is object of DOM.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans: We can select html elements in 4 way using DOM.

They Are:

  1. getElementById() method.
  2. getElementsByClassName() method.
  3. getElementsByTagName() method.
  4. querySelector() & querySelectorAll() method.

3. What is event bubble?

Ans: Event bubbles are a type of DOM event propagation method. It propagate likes bubbles of water. It first works on the target element. Then acts on its parent element. Then again acts the parent element of parent element. It nested like this. Finally came to the HTML main element and stopped the propagation.

4. What is a callback function and why will you use it?

Ans: When a function used as parameter of another function its called callback function . We use call back function on javascript event handler.
